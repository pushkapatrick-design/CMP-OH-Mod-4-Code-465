Code 465 - CIP Equip/Wires Loose - Not Secured

Example visual placeholder for unraveled lashing wire near CIP support.

Use code four sixty-four when lashing wire used to support CIP wire is unraveled, loose, or broken.

Why it matters

Loose lashing wire creates a contact risk

Lashing wire is used to support CIP wire, but when it becomes unraveled, loose, or broken, it can create the potential to contact SDG&E lines or equipment.

What to include

Required reporting details

Include the CIP name

Record the correct CIP name so the location or asset is clearly identified.

Describe the condition

Note whether the lashing wire is unraveled, loose, or broken and where it is visible.
